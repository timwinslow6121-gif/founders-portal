```python
# Pub/Sub message payload for transcript.fileGenerated event:
# {
#   "transcript": {
#     "name": "conferenceRecords/CONF_ID/transcripts/TRANSCRIPT_ID"
#   }
# }
# After receiving: call Meet REST API to get transcript content:
# GET https://meet.googleapis.com/v2/{transcript.name}/entries

# Pub/Sub pull subscriber pattern:
from google.cloud import pubsub_v1
subscriber = pubsub_v1.SubscriberClient()
subscription_path = subscriber.subscription_path(project_id, subscription_id)
streaming_pull_future = subscriber.subscribe(subscription_path, callback=process_transcript_event)
# callback(message) must call message.ack() to prevent redelivery
```

  <behavior>
    process_transcript_event(message):
    - Decodes message.data (bytes → JSON)
    - Extracts transcript_name from data["transcript"]["name"]
    - Calls Meet REST API to get full transcript entries: GET https://meet.googleapis.com/v2/{transcript_name}/entries (using google-auth credentials from GOOGLE_APPLICATION_CREDENTIALS)
    - Concatenates all transcript entries into a summary string: "{participantName}: {text}\n"
    - Matching strategy: transcript metadata includes organizer email — look up User by that email, then look for the most recent appointment_scheduled note for that agent within the last 2 hours to find the customer. If no match found, create UnmatchedCall(provider="google_meet").
    - Creates CustomerNote(note_type="meeting_summary", note_text=summary_text, source_url=f"https://meet.google.com/{conference_record_id}", contact_method="video", agent_id=agent.id, customer_id=customer.id, agency_id=agency_id)
    - Calls message.ack() to confirm processing
    - On any exception: message.nack() (triggers Pub/Sub retry) + logs error

    Subscriber loop:
    - Reads GOOGLE_MEET_PUBSUB_SUBSCRIPTION from config
    - streaming_pull_future.result(timeout=None) — runs indefinitely
    - Handles KeyboardInterrupt gracefully

    systemd service (meet_subscriber.service):
    - ExecStart points to the Python interpreter and script path
    - WorkingDirectory = /home/timothywinslowlinux/dev/founders-portal
    - EnvironmentFile = /home/timothywinslowlinux/dev/founders-portal/.env
    - Restart=on-failure, RestartSec=5
    - User=www-data (standard for VPS services)
